Dataset for confirmed and deaths: 
time_series_covid19_confirmed_US and time_series_covid19_deaths_US.csv
“Admin2” = County name
“Province_State” = State name
“Lat” = County Latitude
“Long_” = Longitude
“1/22/20” … “3/20/21” = Confirmed COVID Cases/Deaths on that date (cumulative)

Dataset for Health Insurance: 
SAHIE_21MAR21_16_03_44_68
“Name” = County/State name
“Insured: Number” = Number of insured people
“Insured: %” = Percentage of population insured
